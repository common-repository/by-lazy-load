By Lazy Load for images, videos, iframes. With lightweight script instantly improve your sites load time. Simple use.

== Description ==
By Lazy Load helps improving your sites load time, Images are usually the heaviest downloads on a given webpage and avoiding unloading images that are never seen saves the user bandwidth.

By Lazy Load is lightweight script and very simple to use.

By Lazy Load replaces your post images, gravatar images and content iframes with a placeholder and then loads the contents when the visitor scrolls the page.


== Installation ==
1. Go to Plugins, Add New, Upload Plugin.
2. Upload the ZIP file.
3. Activate the plugin through the 'Plugins' screen in WordPress
